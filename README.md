# DNA -> RNA Calculator

A clean and simple bio-informatics web tool to transcribe DNA sequences into RNA sequences. It includes real-time input validation to prevent incorrect characters.

### 📱 Built on Mobile!
I built this entire project completely on my mobile phone because I don't own a laptop. Coding, testing, and deploying via GitHub on a smartphone was highly challenging, but a great learning experience!

### Features:
- Direct DNA to RNA translation.
- Error validation for invalid bases.
- Mobile-responsive clean UI.
